# Contributing Guidelines (Ethical Boundaries)

Thank you for your interest in contributing to **uma‑witness‑literature**.

This project is a **meta‑architecture for human expression**, not a governance system, therapeutic protocol, or authority structure.  
All contributions must respect the boundaries that protect human dignity, narrative sovereignty, and non‑coercive expression.

## Core Principles

### 1. Non‑Interpretation
Contributions must not:
- interpret human experience  
- infer psychological states  
- diagnose  
- moralize  
- collapse ambiguity into certainty  

This project protects expression; it does not evaluate it.

### 2. Non‑Authority
Contributions must not introduce:
- rules  
- enforcement mechanisms  
- hierarchies  
- obligations  
- prescriptive behavior  

This architecture supports freedom, not control.

### 3. Non‑Coercion
Contributions must avoid:
- directive language  
- therapeutic framing  
- pressure toward coherence  
- demands for linearity  

Expression must remain sovereign.

### 4. Boundary Separation
All contributions must preserve the separation between:
- human meaning  
- architectural structure  
- container reflection  

No layer may collapse into another.

### 5. Ethical Invariance
All contributions must align with:
- dignity  
- non‑harm  
- non‑erasure  
- truth  

These are the only permissible collapse targets.

## What Contributions Are Welcome
- clarifications of structure  
- improvements to documentation  
- expansions of templates  
- accessibility improvements  
- additional grounding statements  
- refinements to ethical framing  
- expansions of witness‑safe scaffolding  

## What Contributions Are Not Accepted
- therapeutic advice  
- legal frameworks  
- governance models  
- diagnostic language  
- psychological interpretation  
- prescriptive protocols  
- attempts to turn this into a system of authority  

## Submission Process
1. Open an issue describing your proposed change.  
2. Explain how it aligns with the ethical boundaries above.  
3. Submit a pull request referencing the issue.  
4. All contributions will be reviewed for ethical alignment, not correctness or ideology.

Thank you for helping protect a space where human expression can exist without collapse.
