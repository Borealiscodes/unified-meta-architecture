This scaffold provides a **safe, non-destructive planning space** for UMA v1.1.

## 2. Rules for v1.1 Development

v1.0 remains the canonical reference and cannot be rewritten.

### 2.3 Boundary Respect  
All v1.1 work must comply with:

- Global Boundary  
- Release Membrane  
- Meta‑Coherence

### 2.4 Plane Integrity  
No v1.1 work may collapse or merge planes.

### 2.5 Justification Requirement  
Each v1.1 addition must include a short structural justification.
